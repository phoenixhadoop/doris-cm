Tool for making Parcel and CSD of DORIS
===

```
bin/doirsbulid.sh - tool for making Parcel and CSD of doris

Usage:
+----------------------------------------+
| ./doirsbulid.sh [--parcel] [--csd] [--all]  |
+----------------------------------------+
parcel: -  build Parcel
csd:    -  build CSD
all:    -  build Parcel and CSD
```

NOTE:
```
1. doris-csd/auxbak Need to rename to aux   doris-csd/aux
2. The execution directory of the script needs to be under bin, otherwise an error will occur
```
